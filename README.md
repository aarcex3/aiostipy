# Aiostipy

A simple framework based on aiohttp inspired by Nestjs modular architecture.\
Also taking some ideas from other python frameworks, e.g., FastAPI ⚡️, Litestar ⭐, Blacksheep 🐑.

_aiohttp + Nestjs + python 🐍 = aiostipy_
